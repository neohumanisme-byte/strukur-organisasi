# Scope of Task

Create a front-end first web application for company organization structure visualization and Job Description access.

## Scope

- Single-page app.
- HTML5, Tailwind CSS CDN, Vanilla JavaScript.
- Static JSON with localStorage persistence.
- Organization tree view.
- Job Description side panel.
- Role simulation: Employee and Admin.
- Admin can add/edit position, edit reporting line, document control, and JD.
- Employee can view and print/download JD.
- Import/export JSON.

## Main Deliverable

One executable `index.html` plus supporting documentation for Codex.
