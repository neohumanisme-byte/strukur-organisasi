# Front-End Specification

## Stack

- HTML5
- Tailwind CSS via CDN Play
- Vanilla JavaScript ES6+
- Browser localStorage

## UI Modules

- Header and role switcher.
- Summary cards.
- Toolbar: search, department filter, status filter, expand/collapse, import/export.
- Organization tree.
- Detail side panel.
- Admin create/edit forms.
- Print area.

## Interaction Model

- Click node opens panel.
- Admin button reveals edit forms.
- Save writes to localStorage.
- Import replaces local data after validation.
- Export downloads current data as JSON.
- Print generates JD document.
